  1. 抽象类和接口的区别：

    答：

​ 1. 接口只能包含抽象方法，而抽象类可以包含普通方法。

2. 接口只能定义静态常量，而抽象类即可以定义普通属性。也可以定义静态常量属性。

3.  接口不包含构造方法，抽象类可以包含构造方法。

  1. hashcode和equals的重要性：

    答：

  2. 同一个对象无论何时调用hashcode(),得到的返回值必须一样。

    1. hashcode()返回值相等，对象不一定相同。通过hashcode()和equals()必须唯一确定一个对象。

    2. 一旦重写equals，必须重写hashcode.

6.三种排序

答：

  1. 冒泡排序

    for（int i=0;i<length-1;i++）{

    for(int j=0;j<length-1-i;j++){

    if(array[j]>array[j+1]){

    int temp=array[j];

    array[j]=array[j+1];

    array[j+1]=temp;

    }

    }

    }

  2. 选择排序

    for(int i=0;i<length-1;i++){

    int minIndex=i;

    for(int j=i;j<length;j++){

    if(array[minIndex]>array[j])

    minIndex=j;

    }

    int temp=array[i];

    array[i]=array[minIndex];

    array[minIndex]=temp;

    }

  3. 插入排序(是简单快排中最快的)

    for(int i=1;i<length;i++){

    int temp=array[i];

    int leftIndex=i-1;

    while(leftIndex>=0&&array[leftIndex]>temp){

    array[leftIndex+1]=array[leftIndex];

    leftIndex–;

    }

    array[leftIndex+1]=temp;

    }
